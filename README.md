# pyradio  
Manage internet radio statins for mplayer (can be used with Unix/Linux OS)

## Requirements
python 3.x
mplayer 

## Note  
The internet radio stream URL and other information is written to ~/.url . If the file can't be found. it will be created automatically with the information of SmoothJazz.com.pl (You can delete the stream information later if you want)
